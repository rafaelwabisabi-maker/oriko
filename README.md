# ORIKO

An art-technology project exploring orgone energy through digital and physical media.

## About

ORIKO bridges esoteric energy concepts with contemporary digital art. The project includes interactive web experiences, a networked blessing platform (BLESS.EXE), and physical installations. Submitted to EMAP 2026 as a cross-disciplinary artwork exploring invisible forces and collective intention.

## Tech Stack

- Node.js / Express
- Anthropic Claude SDK
- HTML / CSS / JavaScript
- Vercel (deployment)

## Status

Active. EMAP 2026 submission filed.
